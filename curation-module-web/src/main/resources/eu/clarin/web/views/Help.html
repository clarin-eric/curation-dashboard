Curation Module

Curation Module is a project founded by CLARIN EU project. Its goal is to help CMD meta-data authors and curators improving meta-data quality. The curation More information about CMD framework can be found here. The application analyses individual CMD profiles, records and whole collections according to a number of quality criteria and generates statistics in the XML format which can be used by humans or machines to discover potential problems leading to a lower quality. The web application allows users to examine individual profiles and records via web form or via RESTful API. On the "Public Profiles" view users can find information about preprocessed public CMD profiles. Also there is a link to the SMC Browser in which users can explore the structure of CMD profiles and components. "Collection" view contains statistical information about preprocessed collections. Code can be found on github.

Usage

The web form accepts as an input for the instances curation an URL to CMD record and in case of the profiles curation an URL or an ID in format: clarin.eu:cr1:p_xxxxxxxxxxxxx

Rest API

Profile Curation

rest/profile/id/clarin.eu:cr1:p_xxxxxxxxxxxxx

rest/profile/?url=URL

Instance Curation

rest/instance/?url=URL
